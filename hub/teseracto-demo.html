Animación SVG del isotipo Workalógico — Idéntico al logo original

0.3x = Sutil (para headers) • 1x = Normal • 2x = Dramático (splash screens)

Variantes de Animación

Tres estilos de animación optimizados para diferentes contextos. Todas usan el path SVG original del logo para garantizar fidelidad visual.

The Architect

Reveal

Entrada cinematográfica con glow explosivo, scale con rotación 3D, y línea de energía que recorre el contorno. Ideal para landing pages y splash screens.

The Pulse

Loop

Bucle continuo con respiración 3D (scale + rotateX/Y), glow pulsante intenso, y partículas orbitando. Perfecto para headers y estados de carga.

The Energy

Nuevo

Efecto de energía fluyendo con línea de contorno animada, rotación 3D continua suave, y glow pulsante dramático. Para loaders de larga duración.

Código de Uso

JavaScript
// 1. Crear instancia
const teseracto = WoTeseracto.create('#container', {
  variant: 'architect-pulse',  // 'architect', 'pulse', 'energy', o 'architect-pulse'
  color: '#FFCB00',
  intensity: 1.0,           // 0.3 (sutil) a 2.0 (dramático)
  autoPlay: true,
  withParticles: true       // Partículas de energía orbitando
});

// 2. Control de animación
teseracto.pause();          // Pausar
teseracto.play();           // Continuar
teseracto.restart();        // Reiniciar desde el inicio
teseracto.setIntensity(1.5); // Cambiar intensidad en vivo
teseracto.destroy();        // Eliminar

// 3. Animaciones individuales
teseracto.animateArchitect({ duration: 2500 });
teseracto.animatePulse({ intensity: 1.5 });
teseracto.animateEnergy({ intensity: 2 });
