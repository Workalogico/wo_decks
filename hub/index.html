Sistema de Temas

El Design System soporta dos temas intercambiables. Usa el switch animado de arriba o el atributo data-wo-theme en el body.

Tema Amarillo (Brand) Default
<body data-wo-theme="yellow">
1°
2°

Amarillo protagonista, azul secundario

Tema Azul (Tech) Técnico
<body data-wo-theme="blue">
1°
2°

Azul protagonista, amarillo secundario

Variables Semánticas

CSS 
/* Cambian según el tema activo */
var(--wo-accent-primary)       /* Color protagonista */
var(--wo-accent-secondary)     /* Color secundario */
var(--wo-accent-dim)           /* Fondo sutil primary */

/* Clases CSS */
.accent       /* Texto en color protagonista */
.accent-alt   /* Texto en color secundario */
