Mapbox Theme

Tema de mapa oscuro optimizado para visualizaciones de inteligencia geoespacial. Alineado con el Design System Workalógico.

Dark Theme WCAG AA CVD-Friendly 3D Buildings Spanish Labels Geointeligencia

Accesibilidad v2.1: Este tema cumple con WCAG AA (contraste mínimo 4.5:1) y utiliza una paleta CVD-friendly que evita combinaciones rojo-verde problemáticas para daltonismo.

Accesibilidad

Principios aplicados para cumplir con WCAG AA y ser accesible para daltonismo.

WCAG AA Compliance

  • Contraste mínimo 4.5:1 en todas las etiquetas
  • Halos de texto más opacos (0.9) para mejor separación
  • Etiquetas de ciudad: 15.3:1 ratio
  • Etiquetas de calle: 4.6:1 ratio (justo AA)

CVD-Friendly (Daltonismo)

  • Evita rojo-verde en escalas de scoring
  • Usa Indigo→Amarillo para datos secuenciales
  • Naranja en lugar de rojo para alertas
  • Cyan vs Azul para diferenciación de POIs

Uso

Opción 1: Componente WoMapbox

Usar el componente integrado con el tema por defecto.

JavaScript 
// El tema 'workalogico' es el default
const map = new WoMapbox('map-container', {
  center: [-99.1332, 19.4326], // CDMX
  zoom: 12,
  pitch: 45,
  style: 'workalogico' // Tema Workalógico (default)
});

// También puedes usar el estilo estándar de Mapbox
const mapStandard = new WoMapbox('map-container', {
  style: 'mapbox://styles/mapbox/dark-v11'
});

Opción 2: Mapbox GL JS Directo

Usar el estilo JSON directamente con Mapbox GL JS.

JavaScript 
// Usar la constante WO_MAPBOX_STYLE exportada
mapboxgl.accessToken = 'YOUR_TOKEN';

const map = new mapboxgl.Map({
  container: 'map',
  style: WO_MAPBOX_STYLE, // Tema Workalógico
  center: [-99.1332, 19.4326],
  zoom: 12,
  pitch: 45,
  antialias: true
});

HTML Básico

Agregar Capas de Datos

Ejemplos para agregar datos personalizados manteniendo la estética del tema.

Puntos con Scoring

JavaScript 
// Agregar fuente GeoJSON
map.addSource('locations', {
  type: 'geojson',
  data: myGeoJSON
});

// Capa de puntos con color por score (colores de marca)
map.addLayer({
  id: 'location-points',
  type: 'circle',
  source: 'locations',
  paint: {
    'circle-color': [
      'interpolate', ['linear'], ['get', 'score'],
      0, '#FF6B6B',   // --wo-danger
      50, '#FFCB00',  // --wo-yellow-spark
      100, '#10B981'  // --wo-success
    ],
    'circle-radius': 10,
    'circle-stroke-color': '#F7F7F7',
    'circle-stroke-width': 2
  }
});

Heatmap de Densidad

JavaScript 
map.addLayer({
  id: 'density-heat',
  type: 'heatmap',
  source: 'locations',
  paint: {
    'heatmap-color': [
      'interpolate', ['linear'], ['heatmap-density'],
      0, 'rgba(15, 15, 26, 0)',        // Transparente
      0.2, 'rgba(89, 104, 234, 0.3)',  // Blue Lab (bajo)
      0.5, 'rgba(89, 104, 234, 0.6)',  // Blue Lab (medio)
      0.8, 'rgba(255, 203, 0, 0.8)',   // Yellow Spark (alto)
      1, '#FFCB00'                      // Yellow Spark (máx)
    ],
    'heatmap-radius': 25,
    'heatmap-opacity': 0.8
  }
}, 'poi-retail-highlight'); // Insertar debajo de POIs
