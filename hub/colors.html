Uso Recomendado

  • Yellow Spark: CTAs, highlights, badges especiales, estados activos
  • Blue Lab: Accent primario, enlaces, datos clave, hover states
  • Black Core: Fondos principales, superficies elevadas (dark theme)
  • White Focus: Texto/logos sobre fondos oscuros
  • Gray Dark: Texto/logos sobre fondos claros (nuevo 2026)

Ejemplo de Uso

CSS 
/* Usar variables CSS en tus estilos */
.mi-componente {
  background: var(--wo-dark-surface);
  color: var(--wo-text);
  border: 1px solid var(--border-accent);
}

.mi-componente:hover {
  border-color: var(--wo-blue-lab);
}

.mi-componente.destacado {
  background: var(--wo-yellow-dim);
  border-color: var(--wo-yellow-spark);
}
