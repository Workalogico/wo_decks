Paleta de Colores

Sistema cromático de Workalógico. Click en cualquier color para copiar su variable CSS.

Colores de Marca

Los 4 colores principales que definen la identidad visual de Workalógico.

Yellow Spark
--wo-yellow-spark
#FFCB00
Blue Lab
--wo-blue-lab
#5968EA
Black Core
--wo-black-core
#1A1A2E
White Focus
--wo-white-focus
#F7F7F7

Uso Recomendado

  • Yellow Spark: CTAs, highlights, badges especiales, estados activos
  • Blue Lab: Accent primario, enlaces, datos clave, hover states
  • Black Core: Fondos principales, superficies elevadas
  • White Focus: Texto principal en modo oscuro

Ejemplo de Uso

CSS 
/* Usar variables CSS en tus estilos */
.mi-componente {
  background: var(--wo-dark-surface);
  color: var(--wo-text);
  border: 1px solid var(--border-accent);
}

.mi-componente:hover {
  border-color: var(--wo-blue-lab);
}

.mi-componente.destacado {
  background: var(--wo-yellow-dim);
  border-color: var(--wo-yellow-spark);
}
