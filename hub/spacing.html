Sistema de Espaciado

Escala de espaciado consistente optimizada para presentaciones Reveal.js en viewport 1280x720.

Escala de Espaciado

Valores de espaciado progresivos para padding, margin y gaps.

XS (Extra Small) --space-xs: 0.25rem (4px)
SM (Small) --space-sm: 0.5rem (8px)
MD (Medium) --space-md: 0.75rem (12px)
LG (Large) --space-lg: 1rem (16px)
XL (Extra Large) --space-xl: 1.5rem (24px)
Gap Card --gap-card: 1em

Border Radius

Valores de redondeo para esquinas de componentes.

SM
4px
MD
8px
LG
12px
XL
16px
FULL (Pills/Badges)
50px

Uso Recomendado

Padding de Cards

  • 0.6em 0.8em - Cards compactas
  • 1.2em 1.8em - Métricas grandes
  • 0.8em 1.2em - Quotes

Gaps en Grids

  • 0.5em - Grids de cards
  • 0.6em - Stats row / Metrics row
  • 0.8em - Columns layout

Ejemplo de Uso

CSS 
/* Usar variables de espaciado */
.mi-card {
  padding: var(--space-lg) var(--space-xl);
  margin-bottom: var(--space-md);
  border-radius: var(--radius-md);
}

.mi-grid {
  display: grid;
  gap: var(--gap-card);
}

.mi-badge {
  padding: var(--space-xs) var(--space-sm);
  border-radius: var(--radius-full);
}
