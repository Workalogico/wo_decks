Particles System

Sistema de fondos animados con tsParticles para presentaciones Reveal.js. Concepto visual "Data Network" que representa inteligencia de datos conectada.

Vista Previa

Red de nodos conectados con movimiento sutil y colores de marca.

Data Network Effect tsParticles

Archivos del Sistema

wo-particles.css Estilos

Contiene el contenedor #tsparticles, capas z-index, y efectos de viñeta. Incluye variantes por servicio y responsive.

wo-particles.js Config

Configuración de tsParticles con detección automática de tema, variantes por servicio, y integración con Reveal.js.

Instalación

1. Incluir dependencias

HTML (head) 
<!-- Workalógico Particles CSS -->
<link rel="stylesheet" href="../wo-particles.css">

<!-- tsParticles CDN -->
<script src="https://cdn.jsdelivr.net/npm/tsparticles@2/tsparticles.bundle.min.js"></script>

<!-- Workalógico Particles Config -->
<script src="../wo-particles.js"></script>

2. Agregar contenedor

HTML (body) 
<!-- Contenedor de partículas (ANTES de .reveal) -->
<div id="tsparticles"></div>

<div class="reveal">
  <div class="slides">
    ...
  </div>
</div>

3. Activar en slides específicos

HTML 
<!-- Portada con partículas -->
<section data-background-gradient="..." data-particles="true">
  ...
</section>

<!-- Slide de contenido SIN partículas -->
<section data-background-color="#0F0F1A">
  ...
</section>

<!-- CTA con partículas -->
<section data-background-gradient="..." data-particles="true">
  ...
</section>

4. Inicializar con Reveal.js

JavaScript 
Reveal.initialize({
  // ... configuración Reveal.js
}).then(() => {
  // Inicializar partículas con variante (opcional)
  setupWoParticles('enterprise');
});

Variantes por Servicio

Configuraciones predefinidas para diferentes tipos de presentaciones.

geo
Más puntos de datos, conexiones amplias
60 partículas, 150px links
ai
Balance 50/50 de colores, más conexiones
55 partículas, opacity 0.15
marketing
Más dinámico y energético
45 partículas, speed 0.8
enterprise
Más estructurado y elegante
40 partículas, speed 0.5
JavaScript 
// Usar variante específica
setupWoParticles('geo');      // Geointeligencia
setupWoParticles('ai');       // AI Experts
setupWoParticles('marketing');// Marketing
setupWoParticles('enterprise');// CRM/Enterprise

// Sin variante (usa configuración default)
setupWoParticles();

Capas Z-Index

Orden de renderizado de las capas visuales.

Arquitectura de Capas 
z-index: 20  →  .reveal .slides (contenido)
z-index: 10  →  #tsparticles (partículas)
z-index: 5   →  body::after (viñeta)
z-index: 1   →  .reveal .backgrounds (fondos)

Slides Recomendados

Usar partículas solo en slides clave para no distraer del contenido:

  • Portada (Slide 1) - Primera impresión impactante
  • Secciones/Transiciones - Divisores visuales entre bloques
  • CTA/Cierre - Refuerzo del llamado a la acción

Tip: No usar en slides de contenido denso (cards, tablas, métricas) - las partículas pueden competir con la información.
