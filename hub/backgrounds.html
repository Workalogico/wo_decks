Fondos de Slides

Gradientes y patrones para presentaciones Reveal.js. Cada tipo de slide tiene un fondo específico.

Gradientes de Fondo

Tres variantes principales según el tipo de slide.

PORTADA / CTA
Gradient Cover
Slide 1 (portada) y último slide (CTA)
radial-gradient(ellipse at center, #252542 0%, #0F0F1A 70%)
SECCIÓN
Gradient Section
Transiciones entre temas
linear-gradient(135deg, #0F0F1A 0%, #252542 100%)
CONTENIDO
Solid Dark
Slides de contenido estándar
--wo-dark (#0F0F1A)

Grid Pattern

Patrón de cuadrícula sutil que agrega profundidad a los slides de contenido.

Contenido con Grid Background

El patrón de cuadrícula se aplica con la clase .grid-background

CSS 
.grid-background {
  position: absolute;
  top: 0; left: 0; right: 0; bottom: 0;
  background-image: 
    linear-gradient(rgba(89, 104, 234, 0.03) 1px, transparent 1px),
    linear-gradient(90deg, rgba(89, 104, 234, 0.03) 1px, transparent 1px);
  background-size: 40px 40px;
  pointer-events: none;
  opacity: 0.4;
  z-index: -1;
}

CSS - Estilos requeridos 
#tsparticles {
  position: fixed;
  top: 0; left: 0;
  width: 100%; height: 100%;
  z-index: 0;
  pointer-events: none;
  opacity: 0;
  transition: opacity 0.8s ease;
}

#tsparticles.visible { opacity: 1; }

.reveal { position: relative; z-index: 1; }

/* Atenuar grid cuando hay partículas activas */
.particles-active .grid-background { opacity: 0.3 !important; }
JavaScript - Configuración tsParticles 
const particlesConfig = {
  particles: {
    color: { value: ["#5968EA", "#5968EA", "#5968EA", "#FFCB00"] }, // 75% azul, 25% amarillo
    links: {
      enable: true,
      color: "#5968EA",
      distance: 140,
      opacity: 0.12,
      width: 1
    },
    move: {
      enable: true,
      speed: 0.6,
      direction: "none",
      outModes: { default: "out" }
    },
    number: { value: 50 },
    opacity: { value: { min: 0.3, max: 0.7 } },
    size: { value: { min: 2, max: 4 } }
  },
  interactivity: {
    events: {
      onHover: { enable: true, mode: "grab" }
    },
    modes: {
      grab: {
        distance: 180,
        links: { color: "#FFCB00", opacity: 0.5 } // Amarillo al hover
      }
    }
  }
};

// Inicializar
tsParticles.load("tsparticles", particlesConfig);
HTML - Activar en slide 
<!-- Agregar contenedor antes de .reveal -->
<div id="tsparticles"></div>

<!-- Activar partículas con data-attribute -->
<section data-background-gradient="radial-gradient(...)" 
         data-particles="true">
  <div class="wo-cover">...</div>
</section>

<!-- Control de visibilidad en Reveal.js -->
<script>
Reveal.on('slidechanged', event => {
  const particles = document.getElementById('tsparticles');
  const hasParticles = event.currentSlide.dataset.particles === 'true';
  particles.classList.toggle('visible', hasParticles);
  document.body.classList.toggle('particles-active', hasParticles);
});
</script>
CDN - Incluir librería 
<script src="https://cdn.jsdelivr.net/npm/tsparticles@2/tsparticles.bundle.min.js"></script>
