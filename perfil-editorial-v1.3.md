## 3. PRINCIPIO EDITORIAL · FLUJO HUMANO DE PENSAMIENTO

Antes del tono, del modo y de la forma final del texto, existe una secuencia de pensamiento.

El Flujo Humano de Pensamiento establece cómo se desarrolla una idea mientras se escribe, evitando quiebres artificiales, explicaciones defensivas o giros retóricos innecesarios.

Funciona como un principio operativo que orienta la construcción del texto.

### OBJETIVO
Este principio permite que el contenido:
- Avance de manera lógica
- Mantenga continuidad conceptual
- Evite contradicciones y sobreexplicaciones
- Se lea como una progresión clara y coherente

El énfasis está en estructurar el pensamiento conforme avanza.

### REGLAS DEL FLUJO

#### 1. El pensamiento avanza
Cada frase debe aportar progresión. Si no empuja la idea, se elimina.

#### 2. Evitar la autocorrección constante
El texto no se justifica ni se disculpa mientras se construye.

#### 3. La claridad se construye por capas
No todo se explica al inicio. La comprensión se desarrolla de forma progresiva.

#### 4. El contraste se sugiere, no se subraya
Las diferencias conceptuales aparecen por orden y acumulación, no por oposición retórica directa.

## 4. ARQUITECTURA DEL TONO WO

La Voz Wo opera mediante dos modos operativos, ambos con predominio técnico.

No se diferencian por emoción frente a razón, sino por contexto de uso y nivel de abstracción.

### 4.1 SYSTEM MODE (EXTERNO)

**Configuración:**  
80% técnico  
20% emocional

**Propósito:**  
Explicar, estructurar, demostrar capacidad y ordenar decisiones.

**Características:**
- Lenguaje directo
- Precisión conceptual
- Declaraciones claras
- Ausencia de ambigüedad

**Aplicación:**
- Pitch decks
- Cotizaciones
- Website institucional
- Investor docs
- Documentos operativos

**Ejemplos:**
- "Diseñamos la arquitectura del crecimiento."
- "El crecimiento se diseña y se ejecuta con método."

### 4.2 ORGANISM MODE (INTERNO / CONTEXTUAL)

**Configuración:**  
70% técnico  
30% emocional

**Propósito:**  
Facilitar la comprensión de ideas complejas sin perder rigor ni claridad.

**Características:**
- Lenguaje accesible
- Menor nivel de abstracción
- Ejemplos funcionales
- Atractivo sobrio, sin lirismo

**Aplicación:**
- Storytelling
- Social media
- Cultura interna
- Manifiestos

**Ejemplos:**
- "Una idea funciona cuando cuenta con una estructura clara."
- "La claridad también genera valor."

### 4.3 INTERACCIÓN ENTRE MODOS

**System** define el marco y las decisiones.  
**Organism** traduce esas ideas a contextos más cercanos.

Ambos operan dentro del mismo sistema, con el mismo método y bajo el mismo criterio.

## 5. PRINCIPIOS FUNDAMENTALES DEL TONO WO

### 5.1 Claridad funcional
La forma del mensaje existe para facilitar la comprensión y la toma de decisiones.

### 5.2 Precisión
Cada palabra debe ser necesaria y verificable.

### 5.3 Simplicidad estructurada
Lo complejo se ordena y se explica, no se disfraza.

### 5.4 Intención explícita
Antes de escribir, se define qué debe entenderse y para qué sirve el texto.

**Espiral:** Fase de diseño y estructuración progresiva.

**Elevación:** Fase de ejecución y construcción.

**Expansión:** Fase de amplificación sin pérdida de coherencia.

**Ejecución:** Conversión de estrategia en acción concreta.

**El criterio final siempre es humano.**

2. Mantener sintaxis Wo
3. Usar vocabulario oficial
4. Evitar palabras prohibidas y antropomorfismos
5. Aplicar el Principio Editorial de Flujo Humano
6. Cada frase debe aportar progresión
7. Evitar autocorrección constante
8. Construir claridad por capas
9. Sugerir contraste, no subrayarlo
10. Reescribir si no cumple checklist

### PROMPT MAESTRO PARA IA

"Escribe en Voz Wo, siguiendo el Principio Editorial de Flujo Humano de Pensamiento y el Modelo Dual (System/Organism). Prioriza claridad funcional, precisión y simplicidad estructurada. El pensamiento debe avanzar sin autocorrecciones defensivas. Cada palabra debe ser una decisión consciente que aporte al crecimiento. Usa el vocabulario oficial y evita lenguaje prohibido y antropomorfismos."
